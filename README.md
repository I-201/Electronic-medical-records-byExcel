## Description
エクセル版　電子カルテ

Electronic-medical-records-byExcel/blob/master/顧客/名前（テンプレ）_ナマエ.xls
をコピー＆ペーストしカルテを作成

Electronic-medical-records-byExcel/顧客一覧.xlsm
から顧客一覧の更新＆一覧からそれぞれの顧客カルテへ移動可能

使用方法はDemo参照
